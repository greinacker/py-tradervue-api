__init__(self, username, password, user_agent, target_user = None, baseurl = 'https://www.tradervue.com', verbose_http = False)

Construct a TraderVue instance.

Parameters:
  • username (str) – the TraderVue username
  • password (str) – the TraderVue password
  • user_agent (str) – the user agent to use in requests. Should be something like: MyApp (your@email.com)
  • or None target_user (str) – the user id to issues requests on behalf of. To be used by organization administrators (if the feature is enabled)
  • baseurl (str) – the organization’s URL if using a local server
  • verbose_http (bool) – set to True for verbose dumping of HTTP requests and reponses (requires logging of DEBUG severity to be enabled)

:return the TraderVue instance :rtype: TraderVue
